#  633. 平方数之和    

> 链接：https://leetcode-cn.com/problems/sum-of-square-numbers          

给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a*a + b*b = c。    

示例1:  
> 输入: 5  
输出: true  
解释: 1 * 1 + 2 * 2 = 5  

示例2:  
> 输入: 3  
输出: false
