#  387. 字符串中的第一个唯一字符
> https://leetcode-cn.com/problems/first-unique-character-in-a-string/

给定一个字符串，找到它的第一个不重复的字符，并返回它的索引。如果不存在，则返回 -1。

案例:  
> s = "leetcode"  
返回 0.  
s = "loveleetcode",  
返回 2.
